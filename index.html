NPM Package (hk-mahjong)

This is the first-ever NPM package that implements the scoring rules of Hong Kong Mahjong. The package applies the Depth First Search algorithm to solve complex problems in the Mahjong context. The package is written in Typescript with the Object-oriented programming approach. Jest framework is adopted for unit testing.
